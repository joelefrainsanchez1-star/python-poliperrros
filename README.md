# python-poliperrros
podemos:
registrar nuevos pliperros
Ver un menu desplegable de todos lo poliperros
Encontrar a un poliperro
Eliminar un poliperro registrado

Si el poliperro no cuenta con una imagen, el sistema asigna una imagen por defecto (dog.png).
los datos se almacenan:

En memoria (listas del programa)

En un archivo de texto (poliperros.txt) para respaldo
